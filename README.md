# Ages Of Siege Pack

This repository contains the modpack and design work for `Ages Of Siege`.

## Contents

- `docs/`
  - design documents
- `modpack/`
  - packwiz source
  - exported `.mrpack`
  - Prism import helper
- `tools/packwiz/`
  - local packwiz utility used to manage the pack

## Current Target

- Minecraft `1.20.1`
- Fabric
- Phase 1 shortlist pack

## Notes

The custom Fabric gameplay mod remains in a separate project/repository.

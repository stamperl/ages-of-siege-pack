# Ages Of Siege Modpack

This folder contains the `packwiz` modpack source for the first playable `Ages Of Siege` pack target.

## Current Target

- Minecraft `1.20.1`
- Fabric `0.18.4`
- Phase 1 locked shortlist installed

## Included Categories

- foundation
- QoL
- performance
- gameplay backbone

## Current Gameplay Stack

- Origins
- Modern Industrialization
- Guard Villagers
- Illager Invasion

## Current QoL / Performance Stack

- EMI
- Jade
- Inventory Profiles Next
- Sodium
- Lithium
- ModernFix
- FerriteCore
- Dynamic FPS

## Important Note

The custom `Ages Of Siege` control mod is not wired into this pack yet. This modpack currently represents the base stack we will build around.
